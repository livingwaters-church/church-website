# Living Waters Community Church Website

Welcome to the official repository for the Living Waters Community Church website makeover. This project modernizes our digital presence, transitioning from an legacy platform to a streamlined, ultra-fast static architecture hosted completely on GitHub Pages.

## 📌 Project Goals
* **Consolidation:** Reduce the historic 23-page clutter into a tight, manageable 6-page architecture.
* **Modern Design:** Implement a clean, responsive layout featuring card-based components and high-contrast, professional typography.
* **Cost-Efficiency:** Leverage GitHub Pages for secure, $0/month production hosting.

## 🗂️ Site Architecture & Directory Map
* `/` -> `index.html` (Home Page: Welcome Hero, Service Times, Announcements)
* `/about/` -> `index.html` (Visit Us: Service times, mapping, logistics)
* `/about/` -> `beliefs.html` (Our Beliefs: Vision, doctrine)
* `/about/` -> `staff.html` (Leadership/Staff profiles)
* `/ministries/` -> `index.html` (Unified Filter Grid: Kids, Students, Men, Connection Groups)
* `/media/` -> `index.html` (Media Hub: Sermons, Bible Reading Plans, RightNow Media)
* `/donate/` -> `index.html` (Giving Dashboard: General, Benevolence, Mortgage Relief)
* `/assets/` -> Global CSS stylesheets, shared images, and branding vector icons.
